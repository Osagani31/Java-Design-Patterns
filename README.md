# ☕ Java & Object-Oriented Programming – Learning Repository

Welcome to my **Java + OOP Learning Journey** 🚀  
This repository contains notes, examples, and mini-projects I’ve completed while learning **Java fundamentals** and **Object-Oriented Programming concepts**.

---

## 📚 Overview
This repo is designed to help beginners and intermediate learners master Java programming while applying **core OOP principles** in real-world examples.

---

## 🧠 Learning Goals
- Understand Java syntax, data types, and control flow.
- Master the **four pillars of OOP**:
  1. **Encapsulation** – Data hiding using classes and private variables.  
  2. **Inheritance** – Reusing and extending existing classes.  
  3. **Polymorphism** – One interface, multiple implementations.  
  4. **Abstraction** – Simplifying complex systems using abstract classes & interfaces.
- Implement clean, reusable, and modular code.
- Build mini-projects using OOP design.
